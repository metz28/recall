# Sample Machine Learning Documents

This directory contains sample documents about various machine learning topics for testing the Recall knowledge base application.

## Files

### 1. intro_to_machine_learning.md (3.2 KB)
Introduction to machine learning fundamentals covering:
- Types of ML (supervised, unsupervised, reinforcement)
- Key concepts (training/testing, overfitting, bias-variance)
- ML pipeline and popular libraries

### 2. neural_networks_basics.txt (5.5 KB)
Comprehensive guide to neural networks including:
- Architecture components (layers, neurons)
- Activation functions (ReLU, Sigmoid, Tanh)
- Forward and backward propagation
- Training techniques and best practices

### 3. deep_learning_fundamentals.md (8.9 KB)
Deep dive into deep learning covering:
- CNN, RNN, Transformer architectures
- Training strategies and optimization
- Transfer learning
- Popular frameworks (TensorFlow, PyTorch)

### 4. natural_language_processing.txt (11 KB)
Extensive NLP guide covering:
- Core NLP tasks (classification, NER, parsing)
- Text preprocessing and representation
- Word embeddings and transformers
- BERT, GPT, and modern language models

### 5. computer_vision_guide.md (13 KB)
Complete computer vision guide including:
- Image classification, object detection, segmentation
- CNN architectures (ResNet, YOLO, U-Net)
- Data augmentation and transfer learning
- Real-world applications

### 6. reinforcement_learning.txt (13 KB)
Comprehensive RL guide covering:
- Core concepts (agent, environment, policy, value)
- Traditional algorithms (Q-learning, SARSA)
- Deep RL (DQN, PPO, SAC)
- Applications in games, robotics, and more
